# 맥미니 OpenClaw 개인 비서 봇 — 설계서 v1.0.0

> 태그: [확정] 사용자 명시 / [승인] 제안→일괄 승인 / [추론] 기본값 / [TODO] 미해결

## 1. 목표
- OpenClaw 기반 개인 비서 봇. 위임형 비서실장(이한나) + 부서 soul 구조.
- 사용자는 비서실장하고만 대화. 비서실장이 부서에 위임·조율 후 보고.

## 2. 비서실장 자율성·충성
- [확정] 명령권 사용자 단독 잠금 / 최대 자율·최소 개입.
- [확정] 비가역 3종 사전 확인: ① 비용 발생 ② 외부 발송 ③ 파일·데이터 삭제.
- [TODO] 비가역 확인 UX(텍스트 승인/버튼/타임아웃)는 v2. v1 브리핑은 읽기전용이라 미발생.

## 3. 첫 작업 (v1) — 아침 시황 브리핑
- [확정] 발송 시각: 매일 아침 06:00 KST. (전일 미국장 마감 반영 후 시점 — 연중 마감 이후)
- [확정] 커버 스코프: 미국 3대 지수(S&P500·나스닥·다우) 고정 + 워치리스트(v1=반도체). 워치리스트는 config로 두어 섹터 추가 가능 구조.
- [확정] 데이터 성격: 무료 소스로 시작(구체 API는 2순위에서 픽). 무료는 지연·안정성 한계가 있어 송태섭의 "확인 불가" 규칙이 안전망.
- [승인] 입력 소스·언어: 영어권 주요 금융지(뉴스) 입력 → 한국어 출력.
- [확정] 출력 포맷: 시장방향 · 핵심이슈 3(고정) · 주목종목(최대 5) · 리스크 · 한줄결론. 슬랙 한 메시지(길면 스레드).
- [승인] 빈약한 날: 미국 휴장이면 "휴장" 한 줄 / 데이터 일부 결손이면 "확인 불가" 표기 후 진행 / 전체 실패면 이한나가 감독님께 보고.
- [승인] 판단/코드 분리: 수집·파싱·발송 = 무LLM 스크립트, 송태섭 = 선별·해설·한줄결론.

## 4. 채널 / 아키텍처
- [확정] 슬랙 우선. v1은 봇 1개(이한나)만. 송태섭은 별도 봇 아님 — 이한나 내부 위임 에이전트.
- [확정] 단일 봇 내부 위임 구조. (sessions_spawn vs 서브태스크 호출의 구현 디테일은 2순위 Claude Code가 결정)
- [승인] 모델 배정: 이한나·송태섭 판단/해설 = Sonnet(운전대), 대량 분류·요약 = Haiku(잔일), 수집·파싱·발송 = 스크립트(무LLM).
- [승인] 크론: OpenClaw 내장 cron 1일 1회(없으면 맥 launchd). 하트비트 OFF.
- 워크플로우: 크론 → 이한나 기상 → 송태섭 위임 → 수집·선별·해설 → 포맷 → 이한나 종합 → 슬랙 발송.

## 5. 서브에이전트 "직접 호출 금지"
- [확정] 제어권 체이닝 금지(무한루프·맥락 증발 방지). 토론은 허용.
- [승인] 기본 사회자 방식 A(이한나 경유, 라운드 상한 2~3). 방식 B(공유 스레드)는 선택. v1은 부서 1개라 미적용.

## 6. 리스크 / 리얼리티 체크  (설치 전 준비 추가)

### 6.x 설치 전 준비 (보안 기본)
- [승인] 봇 전용 macOS 계정 분리: 맥미니 본계정이 아니라 봇 전용 사용자 계정을 새로 만들어 거기서만 OpenClaw 구동. OpenClaw가 머신 제어권을 갖는 만큼, 같은 기기라도 개인정보·키체인·기존 파일과 권한/파일을 격리한다.
- [승인] 비밀 관리: Anthropic 키·슬랙 토큰(xoxb/xapp)은 openclaw.json에 평문 저장됨 → openclaw.json·workspace를 .gitignore로 git 제외. 공개 repo에는 설계서·soul 텍스트만 올린다. 키는 기기에만.
- [승인] 데이터 보존: 실행 로그·브리핑 산출물은 로컬에 누적(민감정보 없음). 3순위 실측(토큰·비용)을 위해 필요. 정리 정책은 v2.

## 7. 비용 설계 (24시간 운영)
- [확정] 하트비트 OFF, 크론 1일 1회. 유휴 0원이 목표.
- [승인] 실행당 상한: 1회 기사 ≤ 15개, 1회 토큰 상한 설정, 일 비용 상한 설정 → 초과 시 이한나 정지·보고.
- [승인] 컨텍스트 다이어트(긴 기억 미부착·당일 도구만) + 고정 포맷 프롬프트 캐싱.
- [확정] 두뇌 경로: 종량제(Anthropic API)로 시작.
- [승인] 정액 전환 트리거: 2주 실측 후 월 환산이 GLM 정액(~$10)의 2배 초과면 전환 검토.
